# lv-runner (Claude Code 스킬)

> **참고**: 이건 **Claude Code 전용 스킬** 파일이다. **Codex는 스킬 개념이 없고 `AGENTS.md`를 읽어 실행**한다. 두 문서의 워크플로우는 동일하다.

## 워크플로우
1. `npm install` (최초 1회, 또는 `npm run setup`으로 필요 프로그램까지 점검)
2. `npm run launch` — 실제 브라우저를 띄운다. **사용자가 직접 로그인**(자동 로그인 금지: IdP가 차단). 대상 과목을 한 번 열어 활성화.
3. `npm run enumerate` — 주차/항목/진행률을 `.state/report.json`에 수집.
4. `npm run auto -- <주차들>` — 1배속 완주 + 완료처리. 마감 임박 주차를 **앞 순서**로.
5. `npm run report` — 진행률/완료 상태 확인.

## 규칙
- **1배속만** 인정(실제 경과시간 집계). 완료처리(`#close_`) 클릭 + 인정기간 내라야 점수 반영.
- 본인 계정·본인 수강 항목만. 로그인은 항상 사람이 직접.

상세 동작 원리·셀렉터·트러블슈팅은 [`AGENTS.md`](./AGENTS.md) 참조.
